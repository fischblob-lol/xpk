**xpk** is a lightweight, fast, and modern package manager written in
**Zig**.

Built around highly secure repositories, signed metadata, and reproducible
packages.


Features
--------

- Written in Zig  
- 0 Runtime Dependencies
- Repository indexing  
- Simple ``xbuild`` package format  
- Fast native binaries, available for many operating systems due to zigs cross compiling capabilities
