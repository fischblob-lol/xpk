Getting Started
===============

Building xpk
------------

Clone the repository:

.. code-block:: bash

   git clone https://github.com/fischblob-lol/xpk
   cd xpk


Build:

.. code-block:: bash

   zig build
   or 
   scripts/build.sh # for a slimmer binary 


Test:

.. code-block:: bash

   
   or
   scripts/tests.sh

Running
-------

Check available commands:

.. code-block:: bash

   xpk help
